# library
用struct2+spring+hibernate框架开发的项目，后台数据库采用mysql，该系统的使用对象是图书馆管理人员，具备对图书进行增删改查、借书、还书等基本功能。

使用对象是图书馆管理人员，可以对图书进行添加、修改、删除、查询等操作。可以对图书的借记和归还进行登记,可以进行黑名单的管理。
